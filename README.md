## A01
```
問題文
整数 N が与えられるので、一辺の長さが N であるような正方形の面積を出力するプログラムを作成してください。

制約

```

## A02
```
問題文
N 個の整数 A1,A2,⋯,AN の中に、整数 X が含まれるかどうかを判定するプログラムを作成してください。

制約
・N は 1 以上 100 以下の整数
・X は 1 以上 100 以下の整数
・A1,A2,⋯,AN は 1 以上 100 以下の整数

入力
入力は以下の形式で標準入力から与えられます。
N X
A1 A2 ⋯ AN
 
出力
整数 X が含まれるとき Yes、含まれないとき No と出力してください。
```
